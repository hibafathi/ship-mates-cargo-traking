# 🚢 Ship Mates – Cargo Tracking System

## 📌 Overview

Ship Mates is a web-based cargo tracking system developed to monitor and manage shipments efficiently. The system allows users to track cargo in real time using a unique tracking number and ensures proper management of shipment-related data.

---

## 🎯 Features

* 📦 Generate unique tracking numbers for each shipment
* 👤 Manage customer, item, and receiver details
* 🔍 Real-time cargo tracking system
* 🗄️ Efficient data storage using SQL database
* ⚡ Structured and user-friendly workflow

---

## 🛠️ Tech Stack

* **Frontend:** (if applicable) HTML, CSS
* **Backend:** Java
* **Database:** SQL

---

## ⚙️ How It Works

1. User enters shipment details (customer, item, receiver)
2. System generates a unique tracking ID
3. Data is stored in the database
4. User can track shipment status using the tracking number

---

## 📊 System Design

The application follows a structured approach:

* Data is stored in relational tables
* Each shipment is uniquely identified
* Efficient queries are used for tracking and retrieval
